Microsoft TeamsやSlack等のチャットツールで、席を外していても「離席中」になるのを確実に防ぐためのPythonスクリプトです。

## 💡 開発の背景
マウスを動かすだけですが、環境によってはそれだけでは離席判定を回避できない場合があります。

`TeamsStay` は、「マウスの渦巻き移動」＋「Shiftキー入力」を組み合わせるリレー方式を採用。

OSとアプリの両方にアクティブ信号を送ることで、より確実なステータス維持を実現しました。

## ✨ 主な機能
- **ダブル・アクティブ方式**: マウス移動だけでなく、Shiftキーをエミュレート。
- **マルチプラットフォーム対応**: Windows 11はもちろん、macOSでも動作確認済み。
- **インテリジェント監視**: 5秒ごとにマウスをチェック。あなたが操作している間は動作を停止します。
- **自動終了タイマー**: 業務終了時刻（例 18:00）を設定すれば、定時に自動でプログラムを終了。
- **サウンド通知**: 監視の開始・終了を電子音でお知らせ（マウス作動時は無音で集中を妨げません）。
- **緊急停止機能（Fail-Safe）**: 動作中にマウスを画面の四隅に押し当てると、安全装置が働き即座に終了します。

## 🚀 導入手順

### 1. ライブラリのインストール
Pythonがインストールされた環境で、ターミナル（またはコマンドプロンプト）から以下を実行してください。
```bash
pip install pyautogui
```

### 2. プログラムの実行
```bash
python TeamsStay.py
```

### 3. 設定
起動後、対話形式で以下の設定を行います（Enter連打でデフォルト設定になります）。
1. **待機時間**: 何分間操作がない場合に作動させるか（1〜60分）。
2. **終了時刻**: 何時にプログラムを終了させるか（例 18:00）。
3. **サウンド**: 開始・終了音を鳴らすか。

## 📦 Windowsユーザー向け：ワンクリック起動
毎回コマンドを入力するのが面倒な場合は、以下の内容をメモ帳に貼り付け、`TeamsStay.bat` という名前で保存（スクリプトと同じフォルダ）してください。
ダブルクリックするだけで起動できるようになります。

```batch
@echo off
cd /d %~dp0
python "TeamsStay.py"
pause
```
